# MetalOps - Warehouse Sheet Locator System

## Description
A full-stack warehouse management system to track, locate, and navigate metal sheets using an interactive grid-based warehouse layout.

## Tech Stack
- React.js (Frontend)
- FastAPI (Backend)
- MySQL (Database)
- SQLAlchemy (ORM)

## How to Run

###  Database Setup 

### Backend
cd backend  
python -m uvicorn server:app --reload  

### Frontend
cd frontend
npm install  
npm start  

## Features
- Warehouse grid visualization
- Pathfinding navigation
- Duplicate detection
- Search & filtering
